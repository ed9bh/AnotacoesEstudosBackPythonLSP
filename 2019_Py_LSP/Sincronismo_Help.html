Comando SINCRONISMO

Ferramenta para alinhar dinamicamente um ponto ao longo de um perfil base com o eixo correspondente, destacando a posicao sincronizada em tempo real dentro do AutoCAD.

Visao geral

O comando SINCRONISMO acompanha o deslocamento do cursor sobre um perfil escolhido e projeta essa progressao em um eixo, marcando a posicao equivalente por meio de um circulo temporario. A rotina tambem cria uma linha vertical temporaria para facilitar a referencia visual sobre o perfil.

Preparacao

  • Garanta que o desenho possua um elemento representando o perfil (linha, polilinha, spline ou outro objeto de curva suportado).
  • Tenha disponivel um elemento representando o eixo que sera sincronizado com o perfil.
  • Recomenda-se trabalhar no Model Space para melhor visualizacao das entidades temporarias.

Passo a passo

  1. Execute o comando digitando SINCRONISMO na linha de comando.
  2. Se ja existir uma selecao anterior, escolha entre Manter (usar as mesmas entidades) ou Reiniciar (selecionar novamente).
  3. Quando solicitado, selecione a entidade que representa o perfil base.
  4. Selecione a entidade que representa o eixo que recebera a sincronizacao.
  5. Mova o cursor sobre o perfil. O ponto projetado no eixo sera indicado por um circulo azul temporario.
As entidades temporarias sao removidas automaticamente ao encerrar o comando ou em caso de erro.

Controles e opcoes

Menu inicial

Quando o comando encontra um par de entidades previamente selecionadas, e exibido o prompt:

Selecao[Manter Reiniciar]

  • Manter: reutiliza o ultimo perfil e eixo informados.
  • Reiniciar: permite escolher novas entidades para o perfil e o eixo.

Monitoramento

  • O cursor precisa permanecer dentro da projecao horizontal do perfil para que o ponto sincronizado seja calculado.
  • A linha vertical temporaria acompanha a posicao X do cursor, facilitando a leitura da progressao ao longo do perfil.

Tratamento de erros

  • Se ocorrer alguma interrupcao, o comando limpa as entidades temporarias antes de finalizar.
  • Uma mensagem no prompt de comando indica o erro ocorrido para facilitar o diagnostico.

Dicas e boas praticas

  • Mantenha o perfil e o eixo com comprimentos equivalentes ou proporcionais para resultados mais previsiveis.
  • Ajuste a escala do desenho para que o circulo temporario de raio 5.0 seja facilmente visivel.
  • Use camadas especificas para as entidades de perfil e eixo, facilitando a selecao quando trabalhar com desenhos complexos.
